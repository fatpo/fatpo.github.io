## 创建局域网

```
docker network create mynetwork
```

## 查看局域网

```text
docker network ls
```

## 使用局域网
假设现在有 3 台容器，nginx，mysql和 java，想让它们 3 个使用同一个局域网：

```text
docker run --name mysql-container --network mynetwork -e MYSQL_ROOT_PASSWORD=123qwe -d mysql

docker run --name ng -v /Users/fatpo/Desktop:/tmp/ -d --network mynetwork -p 10000:80 nginx

docker run --name jdk8 -v /Users/fatpo/Desktop:/tmp/ -it -d --network mynetwork openjdk:8 /bin/sh
```
